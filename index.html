About Project

Welcome to your first major HTML project! You'll be building a "Student Profile & Course Portal."
This is a multi-page website designed to act as a personal hub for a student,
showcasing their profile, courses, grades, and more.

Purpose This Project

The main goal of this project is to understand how websites are structured using HTML.CSS
You'll learn to create different pages, link them together, and present various types of
information using a wide range of HTML tags.
